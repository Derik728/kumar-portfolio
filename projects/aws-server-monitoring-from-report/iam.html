IAM — Groups & Users

Create an IAM Group with least-privilege policies (e.g., CloudWatchReadOnlyAccess). Add a User to the group to view dashboards and alarms without admin permissions.

Steps

  1. Create Group: MonitoringViewers
  2. Attach policy: CloudWatchReadOnlyAccess
  3. Create User, add to group, generate console password
  4. Share sign-in URL and enforce MFA
